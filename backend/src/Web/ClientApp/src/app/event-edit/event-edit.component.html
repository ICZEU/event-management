Bei Veranstaltungen, die keine Konferenz sind (z.B. Konzert, Festival), ist nicht relevant wem das Ticket gehört. Bei Konferenzen benötigen wir auch die genauen Informationen zur Person, der das Ticket gehört. Bei Konferenzen gibt es beim Check-In einen Dialog, während bei anderen Veranstaltungen dem Kontrolleur nur angezeigt wird, ob das Ticket gültig ist oder nicht.
Wenn Besucher den QR-Code auf ihrem Ticket selbst scannen, werden Sie zur Homepage der Veranstaltung weitergeleitet.
Wenn die Option aktiviert ist, werden personenbezogene Information in der URL in Form eines JSON Web Tokens (JWT) mitgesendet. Mehr Informationen
